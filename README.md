# Vale Sombrio PWA v7 - Upload pelo celular

Esta versão foi ajustada para upload manual pelo celular no GitHub.

Todos os arquivos ficam na raiz do repositório. Não precisa criar a pasta assets.

Suba todos os arquivos deste pacote:
- index.html
- manifest.json
- service-worker.js
- README.md
- icon-192.png
- icon-512.png
- todos os arquivos .webp

Depois ative GitHub Pages:
Settings > Pages > Deploy from a branch > main > /root
