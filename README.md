# Vale Sombrio PWA v10

Versão validada para publicação no GitHub Pages.

## Arquivos para subir no GitHub

Suba todos estes arquivos diretamente na raiz do repositório:

- index.html
- manifest.json
- service-worker.js
- README.md
- icon-192.png
- icon-512.png

## Como atualizar no GitHub

1. Entre no repositório `vale-sombrio`.
2. Toque em Add file / Upload files.
3. Suba todos os arquivos deste pacote por cima dos antigos.
4. Clique em Commit changes.
5. Aguarde o GitHub Pages atualizar.

## Observação sobre cache

Como é PWA, o celular pode segurar a versão antiga em cache.
Se isso acontecer, remova o app da tela inicial, abra o link no Chrome/Safari e instale novamente.
